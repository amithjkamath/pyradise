Tutorials
=========

This section provides tutorials for converting some of the most popular radiotherapy datasets. These tutorials provide
a step-by-step guide to convert datasets for deep learning model training. The current tutorials are limited to the
datasets the authors work with, but we welcome contributions from the community to expand the tutorials. If you like to
contribute, we encourage you to generate a `pull request <https://github.com/ubern-mia/pyradise/pulls>`_ on our GitHub repo.
