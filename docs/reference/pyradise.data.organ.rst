Organ Module
============
Module: :mod:`pyradise.data.organ`

.. module:: pyradise.data.organ
    :noindex:

General
-------

The :mod:`~pyradise.data.organ` module provides the functionality to manage information about the :class:`Organ` of a
certain :class:`~pyradise.data.image.SegmentationImage`.

Class Overview
--------------

The following classes are provided by the :mod:`~pyradise.data.organ` module:

+----------------------------------+----------------------------------------------------------------------------------------------------------------------------------------+
| Class                            | Description                                                                                                                            |
+==================================+========================================================================================================================================+
| :class:`Organ`                   | Class to identify an organ on a :class:`~pyradise.data.image.SegmentationImage`.                                                       |
+----------------------------------+----------------------------------------------------------------------------------------------------------------------------------------+
| :class:`OrganRaterCombination`   | Combination class of consisting of an :class:`~pyradise.data.organ.Organ` instance and a :class:`~pyradise.data.rater.Rater` instance. |
+----------------------------------+----------------------------------------------------------------------------------------------------------------------------------------+
